# REFLECT AI Chatbot

A multi-model AI chatbot built using:
- Streamlit
- Gemini API
- Ollama
- Python

Features:
- Cloud LLM support
- Offline local LLM support
- Multi-model switching
- Modern futuristic UI
- Session chat memory

Models:
- Gemini 2.5 Flash
- Phi3 (Ollama)

Tech Stack:
- Python
- Streamlit
- Ollama
- Google Generative AI
